Row

Container component that renders children horizontally with consistent spacing between them. Use Row when you need to arrange components side by side.

Props

PropTypeDefaultDescription
gapnumber8Spacing between child components in pixels

Example

Notes

  • Row is the primary component for horizontal layouts
  • Gap applies spacing between all direct children
  • Children will wrap to the next line if they exceed the container width
