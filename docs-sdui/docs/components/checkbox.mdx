# Checkbox

Checkbox input component that displays a checkbox with a label. Tap to toggle the checked state. Shows a checkmark icon in a filled box when checked. Ideal for forms, filters, terms acceptance, or any boolean input.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `label` | `string` | â€” | Label text displayed next to the checkbox |
| `value` | `boolean` | `false` | Whether the checkbox is checked |

## Notes

- Checkbox displays a checkbox input with a label
- Tap to toggle the checked state
- Checkmark icon appears in a filled box when checked
- Default value is false (unchecked)
- Ideal for forms, filters, terms acceptance, boolean inputs
- Value prop controls the checked state
