# Social Proof

Component that displays trust indicators such as purchase counts, trending badges, or social validation messages. Shows icon plus text or formatted count. Use Social Proof to build trust, show popularity, or highlight social validation on product screens, checkout flows, or any screen requiring credibility indicators.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `text` | `string` | ‚Äî | Text message to display |
| `icon` | `string` | `"üî•"` | Icon emoji or symbol (defaults to fire emoji) |
| `count` | `number` | ‚Äî | Numeric count to display in formatted form |

## Notes

- Shows icon plus text or formatted count
- Use for trust indicators, popularity displays, or social validation
- Common use cases: "1,234 people bought this", "Trending", "Most popular"
- Icon defaults to fire emoji (üî•) if not specified
- Count is formatted with thousands separators (e.g., 1,234)
- Can display text-only, count-only, or combined icon + text + count
- Ideal for product screens, checkout flows, or any screen requiring credibility indicators
