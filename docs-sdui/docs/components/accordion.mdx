# Accordion

Expandable section component that displays a header with an arrow indicator. Tapping the header toggles the visibility of children content. Arrow indicator shows the expanded/collapsed state. Ideal for FAQs, product details, settings, or any collapsible content.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |

| `expanded` | `boolean` | `false` | Whether the accordion is initially expanded |

## Notes

- Accordion displays an expandable section with a header
- Tap header to toggle children visibility
- Arrow indicator shows expanded/collapsed state
- Default expanded state is false (collapsed)
- Ideal for FAQs, product details, settings, collapsible content
- Children are hidden when collapsed and shown when expanded
