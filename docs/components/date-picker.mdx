# Date Picker

Date selection trigger component that displays a button showing the selected date value. Supports date, time, and datetime modes with optional min/max date constraints. In production, connects to native date picker for platform-specific date selection UI.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `label` | `string` | — | Label text displayed on the button |
| `value` | `string` | — | Selected date/time value (ISO 8601 format) |
| `mode` | `"date" \| "time" \| "datetime"` | `"date"` | Selection mode (date only, time only, or both) |
| `minDate` | `string` | — | Minimum selectable date (ISO 8601 format) |
| `maxDate` | `string` | — | Maximum selectable date (ISO 8601 format) |


## Notes

- Date Picker displays a button that triggers date selection
- Button shows the selected value or label if no value is set
- Supports three modes: date, time, and datetime
- Min/max date constraints limit selectable range
- Values should be in ISO 8601 format
- In production, connects to native date picker for platform-specific UI
- Default mode is "date"
